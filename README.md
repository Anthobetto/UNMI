# UNMI | Lead Monitoring Solution 🚀

[![Deploy to Render](https://img.shields.io/badge/Deploy-Render-blue?style=flat-square)](https://unmi-ljgw.onrender.com/)

## 📝 Descripción
Solució diseñada para la monitorización y recuperación de leads mediante el control de llamadas no atendidas. El sistema permite optimizar la tasa de conversión y mejorar significativamente el servicio al cliente de forma automatizada.

## 🛠️ Stack Tecnológico
* **Lenguaje:** TypeScript para una arquitectura robusta y tipada.
* **Backend:** Node.js con Express.
* **Base de Datos:** Gestión y almacenamiento mediante Supabase.
* **Frontend:** Desarrollado con React y maquetado con React-bootstrap.
* **Arquitectura:** Implementación de API REST para el registro y seguimiento de incidencias.

## 🚀 Instalación y Ejecución
Para ejecutar este proyecto localmente, sigue estos pasos:

1. **Clonar el repositorio:**
   ```git clone [https://github.com/Anthobetto/UNMI.git](https://github.com/Anthobetto/UNMI.git)```

2. **Instalar dependencias:**
```npm install```

2. **Compilar el proyecto (TypeScript):**
```npm run build```

2. **Iniciar el servidor:**
```node backend/dist/index.js```
